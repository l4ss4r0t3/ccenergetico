# 💡 Calculadora de Consumo Energético

Uma aplicação web simples para calcular o consumo energético de eletrodomésticos e estimar o custo com base na tarifa da distribuidora.

---

## 📋 Sobre o Projeto

Esta calculadora foi desenvolvida para ajudar usuários a entenderem quanto energia seus eletrodomésticos consomem e quanto isso representa financeiramente na conta de luz.

---

## ⚙️ Funcionalidades

- **Calculadora de Consumo:** calcula o consumo em kWh a partir da potência (W) e do tempo de uso (minutos)
- **Calculadora de Custo:** calcula o custo total multiplicando o consumo pelo preço do kWh da distribuidora

---

## 🧮 Fórmulas Utilizadas

**Consumo (kWh):**
```
Consumo = (Potência × Tempo) / 1000
```
> Onde Potência está em Watts e Tempo em minutos.

**Custo (R$):**
```
Custo = Consumo × Preço do kWh
```
